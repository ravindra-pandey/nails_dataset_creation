# Nails Dataset Creation

- This is a dataset for *VATA*, *PITTA*, and *KAPHA* Prediction.
- There are some features of nails that represent a different DOSHA in our body. Hence I have analyzed the dataset according to the different levels of features.
----
## Features that we have considered in this dataset are listed below 

- Nail length
    - [0]: <i> Short </i> 
    - [1]: <i> Normal </i>
    - [2]: <i> Long </i>

- Nail shape
    - [0]: <i> Flat </i> 
    - [1]: <i> Normal </i>
    - [2]: <i> Curvy </i>

- Nail Texture
    - [0]: <i> Shiny </i> 
    - [1]: <i> Normal </i>
    - [2]: <i> Rough </i>

- Moon size
    - [0]: <i> No Moon </i> 
    - [1]: <i> Little Moon </i>
    - [2]: <i> Large Moon </i>

- Color
    - [R] <i> Red </i> 
    - [Y] <i> yellow </i>
    - [W] <i> Whitish </i> 
    - [P] <i> Pale </i> 
    - [C] <i> Cloudy </i>

- Nail Biting
    - [0] <i> No Bitting </i>
    - [1] <i> Bitting </i>

- Nail cuticle texture
    - [0]: <i> Normal </i> 
    - [1]: <i> Bad </i>
    - [2]: <i> Too Bad </i>
